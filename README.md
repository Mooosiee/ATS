## Tech Stack
- React
- Tailwind CSS
- React Router
- Puter (serverless backend & AI)
- Vercel

## Why Resumind?
Resumind was built as a **personal resume analysis tool** to explore how ATS systems evaluate resumes and to reduce dependence on expensive, opaque resume-screening platforms during job and internship applications.

## Target Audience
Students, job seekers, internship applicants
